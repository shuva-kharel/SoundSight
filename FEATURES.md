## Robustness (STEP 4 έΑΦ done)
- **Global graceful-error handler** in `server.py`: any unhandled endpoint error returns a
  spoken-friendly message (HTTP 200), never a 500 έΑΦ verified by forcing a throw. Navigate is a
  separate WebSocket, already wrapped per-frame, so a one-shot failure can't kill it.
- Camera auto-reopen (Pi), model-load fallback (heavyέΗΤlight), network-timeoutέΗΤlocal fallback
  (remote.py), describe/tts internal try-except, and memory caps (`MAX_TRACKS` on tracker /
  announcer / distance-EMA; face track-names time-pruned) are all in place.
